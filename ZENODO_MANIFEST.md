# Zenodo Data Archive — Fe-Mo TCP Phase Dataset

This file documents the data files archived on Zenodo as supplementary material to:

> Forti et al. "Data-efficient machine-learning of complex Fe–Mo intermetallics using domain knowledge of chemistry and crystallography"  
> DOI: **10.5281/zenodo.19427673** — accepted in *npj Computational Materials*

---

## Restoring files from Zenodo

After downloading `FeMo_TCP_dataset.zip` from Zenodo, run:

```bash
python Scripts/setup_from_zenodo.py FeMo_TCP_dataset.zip
```

This restores all pkl/csv files to their correct repository locations so the numbered notebooks run without recomputing expensive descriptors.

---

## Data storage overview

| Location | Contents |
|----------|----------|
| **Git repository** (GitHub) | Notebooks, scripts, Tools package, model hyperparameter JSONs |
| **Zenodo** (`FeMo_TCP_dataset.zip`) | DFT data (JSON), Atoms objects (JSON), all pre-computed descriptors, trained models, prediction outputs |
| **Rebuilt on demand** | `Fe-Mo/Descriptors/PREDICTION__*__*.csv` (SOAP for prediction phases, ~600 MB) — generated by nb09 |

## Contents of `FeMo_TCP_dataset.zip` (~1.4 GB)

### 0. Curated DFT data and Atoms objects (JSON — long-term readable)

Stored as JSON inside the zip (converted from pkl by `Scripts/convert_pkl_to_json_targz.py`).
`setup_from_zenodo.py` converts them back to pkl on first use.

| Zip path | Restored to | Description |
|----------|-------------|-------------|
| `Fe-Mo/FullyCuratedParsedBriefSummary.json` | `Fe-Mo/FullyCuratedParsedBriefSummary.pkl` | Curated DFT results, 291 training structures |
| `Fe-Mo/validation_data/ValidationFullyCuratedParsedBriefSummary.json` | `Fe-Mo/validation_data/ValidationFullyCuratedParsedBriefSummary.pkl` | Curated DFT results, 14 validation structures |
| `Fe-Mo/inchulldft/inchulldft_BriefSummary.json` | `Fe-Mo/inchulldft/BriefSummary.pkl.gz` | Extra validation data from a different HT framework |
| `Fe-Mo/Atomsobjects/Fe-Mo-POSCAR-initial-rescaled-AtomsObjects.json` | `…AtomsObjects.pkl` | ASE Atoms objects, initial geometry |
| `Fe-Mo/Atomsobjects/Fe-Mo-POSCAR-relaxed-all-rescaled-AtomsObjects.json` | `…AtomsObjects.pkl` | ASE Atoms objects, DFT-relaxed geometry |
| `Fe-Mo/Atomsobjects/SUBLATICETAGS_POSCAR-initial.json` | `…POSCAR-initial.pkl` | Sublattice tags for CNAV averaging, initial geometry |
| `Fe-Mo/Atomsobjects/SUBLATICETAGS_POSCAR-relaxed-all.json` | `…POSCAR-relaxed-all.pkl` | Sublattice tags for CNAV averaging, relaxed geometry |

### 2. BOP CN-averaged training descriptors (~14 MB)

Post-processed from raw BOP pkl by `REQUIRE_RAW_DATA_05_ComputeBOPFeatures.ipynb`.
12 CSV files — 2 geometries × 3 BOP models × 2 moment counts (16 / 20).

Pattern: `Fe-Mo/Descriptors/CNAV_parallel_Fe-Mo_{initial|relaxed}_{model}_table_WUBIND_{16|20}.csv`

### 3. ACE descriptors (~18 MB)

9 CSV files covering different many-body order truncations.

Pattern: `Fe-Mo/Descriptors/Fe-Mo-{variant}-ACE-CNAV.csv`

### 4. SOAP descriptors (~55 MB)

5 CSV files for canonical-Fe, canonical-W, and element-specific SOAP parameterisations.

Pattern: `Fe-Mo/Descriptors/soap_features__{species}__{params}.csv`

### 5. Small descriptor pkl files (~0.5 MB)

| File | Description |
|------|-------------|
| `Fe-Mo/Descriptors/DatasetFeatures.pkl` | Per-structure metadata (Phase, Mag, Structure) |
| `Fe-Mo/Descriptors/CNAVPyscal.pkl` | Pyscal coordination-number features |
| `Fe-Mo/Descriptors/matminer_atomic_features.pkl` | Magpie/matminer atomic features |
| `Fe-Mo/Descriptors/CNList.pkl` | CN lookup table |

### 6. Trained model pkl files (~29 MB)

Final KRR ensemble regressors used in nb09/nb11.

| File | Description |
|------|-------------|
| `Fe-Mo/results/voting_regressor_KernelRidge.pkl` | Ensemble VotingRegressor (main model) |
| `Fe-Mo/results/regressors_bag_KernelRidge.pkl` | Bag of individual KRR regressors |
| `Fe-Mo/results/indexed_bag_KernelRidge.pkl` | Index-annotated bag |
| `Fe-Mo/results/KernelRidge_inportances.pkl` | Feature importances |
| `Fe-Mo/results/Fe-Mo_Kernel Ridge_OptimalScores_EF_nmhcp.pkl` | Optimal cross-validation scores |

### 7. Feature-selection learning-curve results (~19 MB)

Forward recursive feature selection outputs, needed to run `07_MachineLearn-ModelSelection.ipynb`.

| File | Size | Description |
|------|------|-------------|
| `Fe-Mo/results/concatenation_results_EF_nmhcp_no_hcp_bcc_fcc_KernelRidge.pkl` | 3.3 MB | KRR learning curves |
| `Fe-Mo/results/concatenation_results_EF_nmhcp_no_hcp_bcc_fcc_MLP.pkl` | 8.4 MB | MLP learning curves |
| `Fe-Mo/results/concatenation_results_EF_nmhcp_no_hcp_bcc_fcc_RandomForest.pkl` | 3.1 MB | Random Forest learning curves |
| `Fe-Mo/results/concatenation_results_EF_nmhcp_FixedOS.pkl` | 2.4 MB | KRR learning curves (fixed onsite variant) |
| `Fe-Mo/results/concatenation_results_EF_nmhcp_FixedNames.pkl` | 1.2 MB | KRR learning curves (fixed names variant) |

### 8. Prediction output CSV files (~84 MB)

Final EF predictions for R, M, P, δ phases across all descriptor/model combinations.
48 files — 4 phases × 3 descriptors × 4 output variants (ensemble, ensemble+1, MAG=0, MAG=1).

Pattern: `Fe-Mo/results/PREDICTION__{phase}__{descriptor}.csv`
